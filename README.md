# 🎵 Blue Neon Audio Visualizer

## 기능 소개

### 1. 오디오 기능
- 음악 파일(`music.mp3`) 로드 및 재생
- Play / Pause / Stop 버튼 제공
- 실시간 오디오 분석 (Amplitude, FFT)

### 2. 시각화 기능
- FULL 모드: EQ + Waveform + Pulse 동시 표현
- EQ 모드: 주파수 기반 막대 그래프
- Waveform 모드: 파형 곡선 시각화
- Pulse 모드: 음량에 반응하는 원형 펄스
- 마우스 클릭 시 Ripple 파동 생성

### 3. 색상 테마
- C 키로 테마 변경 (Blue / Cool)

### 4. 사용자 인터랙션
- Space 키: 시각화 모드 전환
- C 키: 색상 모드 전환
- 마우스 클릭: Ripple 효과 생성

### 5. 반응형 화면
- 창 크기 변경 시 자동으로 Canvas 크기 조정
